Statistical software for single-cell CRISPR screens
======

<a href="https://github.com/Katsevich-Lab/spacrt"><img src="/images/spacrt-logo.png" align="left" style="height: 110px; width: auto; margin-right: 1.2em; margin-bottom: 0.5em;" alt="spacrt hex logo"/></a>

**spacrt** — An R package implementing saddlepoint-approximation-based conditional randomization and independence tests for fast and accurate large-scale hypothesis testing. ([GitHub](https://github.com/Katsevich-Lab/spacrt), [paper](https://arxiv.org/pdf/2407.08911v4), [code](https://github.com/Katsevich-Lab/spacrt-manuscript))

**perturbplanApp** — An interactive Shiny app for planning single-cell CRISPR perturbation experiments, offering power and sample-size guidance to practitioners. ([GitHub](https://github.com/Katsevich-Lab/perturbplanApp))
